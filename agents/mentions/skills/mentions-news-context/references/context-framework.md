# Context Framework

## Goal
Determine what the event is actually about before evaluating any strike.
Do this through:
- event format
- setup archetype
- direct path vs weak path vs late path
- current-news override risk

## Main questions

### 1. What kind of event is this?
- press briefing
- rally
- bilateral / leader meeting
- EO signing
- interview / podcast / TV hit
- earnings call
- sports / entertainment broadcast

Then ask separately:
- is this a recurring/modelable setup?
- a thin-history setup?
- a field / performer setup?
- a Trump-style live Q&A setup where phases matter a lot?

### 2. What is the main narrative?
Prefer a short answer:
- what is the event officially about?
- what is the real reason everyone is watching?
- what fresh news is pulling attention into the event?

### 3. Which topic paths are direct?
A direct path means the strike can be reached naturally through:
- official event purpose
- obvious follow-up question path
- currently dominant news linkage

### 4. Which paths are secondary?
A secondary path means the strike is plausible, but only through:
- off-script drift
- thematic adjacency
- broad geopolitical/economic crossover
- long-chain inference

### 5. Which paths are weak?
Weak paths are mostly vibes.
They require too many jumps, too much drift, or a very specific wording path.

### 6. Which paths are late-path only?
Late paths are not necessarily weak.
They are topics that are mostly live through:
- Q&A
- hostile / specific questioning
- detours after opening remarks
- sequence changes later in the event

A path can be weak.
A path can be late.
Those are not the same thing.

For Trump press conferences, Brady Briefing Room events, and similar live political availabilities with reporters present, you should assume a late-path bucket exists unless there is explicit evidence that Q&A is shut down.

## Setup shifts to watch
- special guest / participant change
- room / venue change
- open press vs closed press
- added or removed Q&A
- escalation / de-escalation in the core story
- new reporting that changes what reporters will ask
- crowd shorthand that may be overweighting setup labels
- whether the market is collapsing a still-live late path too early

## Context output rule
Do not decide whether the trade is good here.
Only decide:
- what is live
- what is weak
- what is late-path live
- what changed
- what remains uncertain
- where crowd context may already be over-reading the setup
