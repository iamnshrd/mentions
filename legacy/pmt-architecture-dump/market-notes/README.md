# Market Notes

Локальный архив разборов рынков под будущий GitHub repo / GitHub Pages.

## Структура
- `notes/` — отдельные разборы по рынкам
- `templates/` — шаблоны для новых разборов
- `index.md` — ручной индекс / витрина

## Правила
- один файл = один market note
- формат: markdown
- investor-style, коротко и по делу
- использовать wording discipline из `wording/markets_wording_db.json`
- strike labels писать human label + monospace, без ticker-spam

## Suggested filename
`YYYY-MM-DD-market-id-short-title.md`

Пример:
`2026-04-06-kxtrumpmentionb-26apr06b-iran-press-conference.md`
