# Output Examples

## Example 1 — Yes lean

- **Setup:** recurring briefing-style market; White House press briefing; recurring format; Q&A likely; topic cluster heavily overlaps with current strike.
- **Lean:** yes
- **Fair value:** 68-76%
- **Why:**
  - the topic is already live in the briefing agenda
  - this spokesperson has used adjacent wording recently
  - Q&A creates an extra path to the mention
  - market appears to be underpricing the direct path relative to generic headline noise
- **Execution:** start with limit orders; take aggressively only if setup shifts further in your favor and price is still lagging
- **Main risks:**
  - topic gets crowded out by a bigger headline
  - Q&A ends quickly
- **Invalidation / no-trade condition:** if price already moves into the high 70s/low 80s before the event starts, edge probably compresses too much

## Example 2 — No lean

- **Setup:** constrained ceremony / announcement format; low odds of Q&A; target topic is only weakly connected to the event reason.
- **Lean:** no
- **Fair value:** 18-26%
- **Why:**
  - weak direct path from event topic to strike
  - no-Q&A setup removes the easiest late path
  - market appears to be pricing generic headline heat rather than actual event structure
  - setup shorthand may be understating how narrow the real path is
- **Execution:** passive no bids preferred; do not chase if no side already steamed
- **Main risks:**
  - speaker goes off-script
  - surprise press interaction creates an extra path
- **Invalidation / no-trade condition:** if the event flips to open press or a fresh news item directly links the strike to the event, reprice from scratch

## Example 3 — No-trade

- **Setup:** thin-history / rare-speaker format; poor comparables; unclear whether Q&A happens; current market already moved a lot.
- **Lean:** no-trade
- **Fair value:** too uncertain to price tightly
- **Why:**
  - weak comparable set
  - setup path is unclear
  - current price already reflects much of the obvious thesis
  - this is fake-precision territory, not a clean edge
- **Execution:** avoid
- **Main risks:**
  - fake precision
  - getting trapped in a bad fill on thin liquidity
- **Invalidation / no-trade condition:** if a concrete setup change appears (guest confirmed, event format clarified, topic path becomes explicit), revisit

## Example 4 — Live execution warning

- **Setup:** event already started; target strike is mainly a late-path / Q&A-path word; live spread is wide.
- **Lean:** conditional yes, but execution-sensitive
- **Fair value:** 40-55% if Q&A stays alive; much lower otherwise
- **Why:**
  - late-question path still exists
  - spread suggests liquidity is withdrawing
  - stale passive orders are dangerous here
  - this is a phase-dependent trade, not a static pre-event number
- **Execution:** either wait for cleaner information or use small aggressive clips; do not leave lazy stale orders up
- **Main risks:**
  - no Q&A
  - instant repricing if the target word is said or ruled out
- **Invalidation / no-trade condition:** if the host/event closes Q&A path or if the spread remains too hostile for clean execution
