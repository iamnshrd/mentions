# Source Priority

## Highest priority
- official event announcements
- official schedules
- official statements / transcripts
- direct reporting from major outlets on the event itself

## Medium priority
- strong beat reporting
- reputable liveblog coverage
- direct quotes summarized by credible outlets

## Lower priority
- generic commentary
- social posts discussing what "might" happen
- recycled takes without fresh reporting

## Use these principles
- prefer event-linked reporting over opinion
- prefer direct quotes over second-hand summaries
- prefer fresh reporting over stale background color
- prefer sources that change the event path over sources that only increase excitement

## Practical rule
A source is most useful when it changes one of these:
- event purpose
- likely topics
- likely questions
- guest list
- format
- venue
- timing
