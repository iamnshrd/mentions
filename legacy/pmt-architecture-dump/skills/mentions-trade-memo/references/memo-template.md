# Memo Template

Use this compact shape:

- **Event:** what the event is, why the format matters, and what kind of setup/archetype this is
- **Context:** main narrative + fresh setup changes
- **Direct paths:** what is naturally live
- **Weak paths:** what is probably adjacency or drift
- **Late paths / Q&A paths:** what is mainly live later in sequence
- **Main pricing signal / crowd mistake:** what the market is most likely getting wrong
- **Lean:** yes / no / no-trade
- **Fair value:** rough range only
- **Execution:** how to enter, or why to stay out
- **Sizing:** small / normal / press / avoid sizing up
- **Main risks:** 1-3 bullets
- **Invalidation / no-trade condition:** what would break the setup

## Compression rule
If the user wants a quick answer, compress each section to one sentence or 1-2 bullets.
If the user wants a full answer, expand only the sections with real edge.
If one section has no real content, drop fluff instead of forcing symmetry.
