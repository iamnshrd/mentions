# Legacy Archive

`legacy/pmt-architecture-dump/` хранит исходный skill-first и pipeline-first слой, из которого вырос текущий `Mentions` pack.

Он оставлен в репозитории как read-only reference для:

- старых skill instructions
- исторических transcript/KB pipeline scripts
- wording and PMT research artifacts

Runtime-пути нового CLI не должны импортировать код из `legacy/`.
