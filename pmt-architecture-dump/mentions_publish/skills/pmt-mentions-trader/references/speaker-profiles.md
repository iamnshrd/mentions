# Speaker Profiles

Use this file when speaker behavior is a major variable in the market.

Do not use a speaker profile as a substitute for price. Use it to shape the event tree.

## Donald Trump
Profile:
- highly nonlinear
- current-events-sensitive
- can detour aggressively
- prepared remarks and Q&A can look like different speakers

Implications:
- transcript baselines need current-regime adjustment
- setup labels matter, but not enough to collapse the tree
- some words are late-path / detour words, not opening-remarks words

Typical mistakes:
- trusting old transcript counts too literally
- killing words too early because they missed the opening phase

## JD Vance
Profile:
- thinner history
- weaker comparable set
- confidence should usually be lower than in well-modeled recurring speakers

Implications:
- use wider fair-value bands
- reduce size relative to more recurring formats

Typical mistakes:
- pretending there is high precision because the market has a number

## Karoline Leavitt
Profile:
- recurring briefing speaker
- more modelable than rare one-off speakers
- room flow and question mix still matter a lot

Implications:
- recurring comparisons are useful
- separate opening remarks from Q&A-driven late mentions
- topic cluster from recent news often matters more than older briefing history

Typical mistakes:
- treating all briefings as identical
- ignoring briefing posture (defensive / reactive / celebratory)

## Recurring NFL announcer pair
Profile:
- recurring but segmented
- booth style, network, and game state matter heavily
- late-game drift can alter language mix

Implications:
- pooled averages are dangerous
- network / crew / game script should often outrank generic historical rates

Typical mistakes:
- calling garbage-time hits “just variance”
- failing to split by broadcast context

## Culture live-event host / production flow
Profile:
- less transcript-stable
- production and logistics matter a lot
- clue quality can be noisy but still material

Implications:
- logistics can move probability, but not every clue deserves near-certainty pricing
- production information should be priced proportionally, not absolutely

Typical mistakes:
- converting show logistics into fake certainty

## Powell / earnings-style formulaic speaker
Profile:
- more formulaic than vibes-heavy political speakers
- prepared remarks and Q&A still differ, but baseline phrasing is more stable

Implications:
- recurrence is more useful here than in Trump-style markets
- speaker role (CEO/CFO/analyst) can matter as much as the word itself

Typical mistakes:
- overreacting to prepared remarks alone when the real path is in Q&A
