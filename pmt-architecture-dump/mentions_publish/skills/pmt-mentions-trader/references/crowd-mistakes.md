# Crowd Mistakes

Use this file when you need to name the market’s error directly.

These are not personal anti-patterns only. These are recurring ways the crowd or order flow misprices the market.

## 1. Overweighting setup labels
- Market turns open press / closed press / room setup into fake certainty.
- Usually strongest before live confirmation.

## 2. Trusting pooled historicals
- Crowd uses broad transcript averages where the real edge requires segmentation.
- Very common in recurring announcer products.

## 3. Copy-trading after the move
- Followers inherit a worse price and confuse the original sharp’s trade with their own worse entry.

## 4. Misreading field information
- Crowd watches pumped names in a field market but fails to reprice the untouched names correctly.

## 5. Assuming setup kills late paths
- Crowd treats early silence or setup shorthand as if late Q&A / detour paths are dead.

## 6. Confusing logistics with probability zero
- Culture / production clues are converted into all-or-nothing certainty instead of probability shifts.

## 7. Assuming random yap is variance, not structure
- In announcer markets, dead-game chatter is treated as “bad luck” instead of a different language regime.

## How to use this file
When diagnosing a trade, try to say explicitly:
- what crowd mistake is happening
- whether price still leaves edge after recognizing it
- what execution follows from that mistake
