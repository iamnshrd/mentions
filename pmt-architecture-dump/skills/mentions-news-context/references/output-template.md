# Output Template

Use this exact structure when possible:

- **Event:** what it is, where, when, what format matters, and what kind of setup this is
- **Main narrative:** 1-3 central themes
- **Fresh changes:** new developments from the last 24-72h
- **Direct paths:** topics or strikes naturally live
- **Weak paths:** adjacency topics that look forced
- **Late paths / Q&A paths:** topics that mainly come alive through questions, detours, or late sequence
- **Q&A watch:** whether question-driven paths matter
- **Probably priced in:** what the market is likely already paying for
- **Unknowns:** unresolved context risks

## Rules
- Keep it compact
- Do not drift into price/execution analysis unless explicitly asked
- If context is still murky, say so
- If the event setup changed materially, emphasize that before everything else
- If setup shorthand looks over-weighted, flag that as a context issue before the trade stage
